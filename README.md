# Elliot v1

Bug bounty automation tool for reconnaissance and vulnerability scanning.

## Features
- Subdomain enumeration
- Subdomain takeover detection
- Live host discovery
- URL crawling
- XSS detection
- Nuclei vulnerability scanning

## Requirements
See requirements.txt

## Installation
git clone https://github.com/username/elliot.git
cd elliot
chmod +x elliot.sh

## Usage
./elliot.sh -h
./elliot.sh -d example.com
